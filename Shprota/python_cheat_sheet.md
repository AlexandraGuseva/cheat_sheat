## 🛠 1. Переменные и базовые типы данных

Переменная — это «коробка» для хранения данных. В Python не нужно указывать тип данных при создании переменной, язык определит его сам.

```python
x = 5                # Целое число (int)
pi = 3.14            # Число с плавающей точкой (float)
name = "Алексей"     # Строка (str) — можно в одинарных ' ' или двойных " " кавычках
is_active = True     # Булево значение (bool) — только True или False (с большой буквы)
```

### Как узнать тип переменной или привести к другому типу:
```python
print(type(x))       # Выведет: <class 'int'>

# Приведение типов (конвертация)
age = int("25")      # Переведет строку "25" в число 25
price = str(99.9)    # Переведет число 99.9 в строку "99.9"
```

## 🧮  3. Математические операции

```python
a = 10
b = 3

print(a + b)   # 13 (Сложение)
print(a - b)   # 7  (Вычитание)
print(a * b)   # 30 (Умножение)
print(a / b)   # 3.3333333333333335 (Обычное деление — всегда дает float)

# Важные специфические операции:
print(a // b)  # 3  (Целочисленное деление — отбрасывает остаток)
print(a % b)   # 1  (Остаток от деления)
print(a ** b)  # 1000 (Возведение в степень, 10 в кубе)
```

## 🔄  4. Условия и логика (if, elif, else)

В Python **отступы (4 пробела)** имеют критическое значение. Они показывают, какой код относится к условию или циклу.

```python
age = 18

if age >= 18:
    print("Доступ разрешен")  # Выполнится, если условие True
elif age == 17:
    print("Подождите еще год") # Выполнится, если первое False, а это True
else:
    print("Доступ запрещен")  # Выполнится во всех остальных случаях
```

### Логические операторы для сложных условий:
* `and` — И (выполняется, если **оба** условия верны).
* `or` — ИЛИ (выполняется, если **хотя бы одно** условие верно).
* `not` — НЕ (инвертирует значение: `not True` станет `False`).

```python
if age >= 18 and is_active:
    print("Пользователь может войти")
```

## 🔁  5. Циклы (Циклические действия)

### Цикл `for` (когда знаем, сколько раз повторить, или обходим элементы):
```python
# range(5) генерирует числа от 0 до 4 (всего 5 штук)
for i in range(5):
    print(f"Итерация {i}")

# Обход букв в строке
for letter in "Python":
    print(letter)
```

### Цикл `while` (выполняется, пока условие истинно):
```python
counter = 0
while counter < 3:
    print(counter)
    counter += 1  # То же самое, что counter = counter + 1
```

## 📦 6. Базовые коллекции (Списки и Словари)

### Списки (List) — упорядоченный набор элементов
```python
fruits = ["яблоко", "банан", "апельсин"]

print(fruits[0])     # "яблоко" (нумерация элементов ВСЕГДА начинается с 0)
print(fruits[-1])    # "апельсин" (минус первый элемент — это последний с конца)

# Изменение и добавление
fruits[1] = "груша"       # Заменит "банан" на "груша"
fruits.append("киви")     # Добавит "киви" в конец списка
fruits.remove("яблоко")   # Удалит элемент из списка

print(len(fruits))        # Длина списка (количество элементов)
```

### Словари (Dict) — хранение данных в формате `ключ: значение`
```python
# Похоже на телефонную книгу
user_info = {
    "name": "Иван",
    "age": 30,
    "city": "Москва"
}

print(user_info["name"])  # Получить значение по ключу -> "Иван"
user_info["age"] = 31     # Изменить значение
user_info["email"] = "ivan@example.com" # Добавить новую пару ключ:значение
```

## ⚙️  7. Функции (Создание своих команд)

Функции нужны, чтобы не копировать один и тот же код многократно.

```python
# Создаем функцию с помощью ключевого слова def
def greet_user(username):
    """Эта функция приветствует пользователя"""
    return f"Привет, {username}!"

# Вызываем функцию и сохраняем результат в переменную
message = greet_user("Анна")
print(message)  # Выведет: Привет, Анна!
```

## 💎  5. Важные фишки, которые часто забывают

### 🪓  1. Срезы (Slices) — как отрезать кусок от строки или списка
Синтаксис: `коллекция[СТАРТ : СТОП : ШАГ]` (правая граница `СТОП` не включается!).

```python
text = "Python"
print(text[0:2])   # "Py" (символы с индексами 0 и 1)
print(text[2:])    # "thon" (от индекса 2 и до самого конца)
print(text[:4])    # "Pyth" (от начала и до индекса 4)
print(text[-2:])   # "on" (последние два символа)

# Разворот строки или списка (шаг -1 идет задом наперед)
print(text[::-1])  # "nohtyP"
```

### 🧠  2. Изменяемые и Неизменяемые типы (Важно!)
В Python типы данных делятся на два лагеря. Это определяет, как они ведут себя при копировании.

* **Неизменяемые** (`int`, `float`, `str`, `bool`, `tuple`): при любом изменении создается *новый* объект.
* **Изменяемые** (`list`, `dict`, `set`): меняются прямо на месте.

```python
# Опасно со списками:
list_a = [1, 2, 3]
list_b = list_a     # Это НЕ копия! Оба имени теперь смотрят на один список.

list_b.append(4)
print(list_a)       # Выведет! Изменив B, мы изменили и A.

# Как сделать ПРАВИЛЬНУЮ копию списка:
list_c = list_a.copy() # Теперь они независимы
```

### 🧰  3. Топ методов строк и списков (Быстрые команды)

#### Для строк (`str`):
```python
s = "  Привет, Python!  "

print(s.strip())        # Убирает пробелы по краям -> "Привет, Python!"
print(s.lower())        # Переводит в нижний регистр -> "  привет, python!  "
print(s.replace("!", "?")) # Заменяет символы -> "  Привет, Python?  "

# Разбить строку в список по разделителю
words = "яблоко,банан,груша".split(",") 
print(words)            # ['яблоко', 'банан', 'груша']

# Склеить список обратно в строку
print("-".join(words))  # "яблоко-банан-груша"
```

#### Для списков (`list`):
```python
lst = [10, 20, 30]

lst.insert(1, 99)   # Вставит число 99 на индекс 1 -> [10, 99, 20, 30]
last = lst.pop()    # Удалит и вернет ПОСЛЕДНИЙ элемент (30)

# Проверка, есть ли элемент в списке (возвращает True/False)
if 20 in lst:
    print("Элемент найден!")
```
