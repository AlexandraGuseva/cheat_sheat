#PyCharm: Горячие клавиши (macOS)

### 🔍 Навигация и Поиск
* `Shift` `Shift` — **Искать везде**: файлы, настройки, действия.
* `Cmd` + `F` / `Cmd` + `R` — Поиск / Замена в текущем файле.
* `Cmd` + `Shift` + `F` — Поиск по всему проекту.
* `Cmd` + `O` — Быстрый поиск класса.
* `Cmd` + `Shift` + `O` — Быстрый поиск файла.
* `Cmd` + `E` — Недавние файлы.
* `Cmd` + `B` (или `Cmd` + клик) — Перейти к объявлению функции/переменной.

### 📝  Редактирование кода
* `Cmd` + `Alt` + `L` — **Форматирование кода** по PEP 8.
* `Opt` + `Enter` — Быстрое исправление ошибок / импорт.
* `Cmd` + `D` — Дублировать строку.
* `Cmd` + `Backspace` — Удалить строку.
* `Cmd` + `/` — Строчный комментарий (`#`).
* `Opt` + `Cmd` + `/` — Блочный комментарий (`"""`).
* `Shift` + `Enter` — Новая строка снизу без разрыва текущей.
* `Opt` + `Shift` + Клик — Мультикурсор.

### ⚡ Рефакторинг, Запуск и Дебаг
* `Shift` + `F6` — **Умное переименование** по всему проекту.
* `Opt` + `Cmd` + `M` — Вынести код в отдельную функцию.
* `Opt` + `Cmd` + `V` — Вынести значение в переменную.
* `Ctrl` + `R` — Запуск текущего скрипта.
* `Ctrl` + `D` — Запуск отладки (Debug).
* `Cmd` + `F8` — Поставить/снять точку останова (Breakpoint).
* `F8` / `F7` — Шаг обхода (Step Over) / Шаг внутрь (Step Into) при дебаге.
