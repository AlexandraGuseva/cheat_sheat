# 🐙  2. Работа с Git (в терминале и PyCharm)

### Горячие клавиши Git в PyCharm
* `Cmd` + `K` — Открыть окно **Commit** (фиксация изменений).
* `Cmd` + `Shift` + `K` — Открыть окно **Push** (отправка в удаленный репозиторий).
* `Cmd` + `T` — **Update Project** (подтянуть изменения из origin / git pull).
* `Ctrl` + `V` — Меню VCS (быстрые операции с Git).

### Консольные команды Git (на всякий случай)
* `git status` — Проверить статус файлов.
* `git add .` — Добавить все изменения в индекс.
* `git commit -m "сообщение"` — Зафиксировать изменения.
* `git checkout -b имя_ветки` — Создать новую ветку и перейти на нее.
* `git merge имя_ветки` — Слить указанную ветку в текущую.
